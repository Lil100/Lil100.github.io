Featured Project

Mifugo360

A comprehensive technology platform transforming livestock management in Africa. Combines IoT, data analytics, and mobile-first design to help farmers manage health, breeding, and market access for their livestock.

Spring Boot React Native PostgreSQL IoT Data Analytics
Problem
African livestock farmers lack integrated tools for tracking animal health, breeding cycles, and accessing fair markets — resulting in losses and inefficiency.
Solution
A unified platform connecting farmers with veterinary insights, market prices, and herd management tools through mobile-first technology.
Impact
Empowering smallholder farmers with data-driven livestock management, improving productivity and market access across Kenya.
Fintech Infrastructure

Digital Banking Platform

A scalable digital banking platform built with Spring Boot microservices architecture. Features account management, transaction processing, loan origination, and real-time notifications — designed for the unbanked and underbanked across Africa.

Spring Boot Microservices Kafka PostgreSQL Docker
Problem
Millions of Africans lack access to reliable digital banking services that are affordable, scalable, and tailored to local needs.
Solution
A microservices-based banking platform with modular services for accounts, transactions, loans, and notifications — serving high transaction volumes reliably.
Architecture
Event-driven microservices with Apache Kafka for async processing, Redis for caching, PostgreSQL for persistence, and containerized deployment.
